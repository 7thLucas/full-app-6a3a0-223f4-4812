Build me a mobile and web application that enables property developers to showcase their projects and available units in a professional and interactive way.

Key Features:

Project portfolio and property listings
Unit catalog with detailed specifications (floor plans, land/building size, bedrooms, bathrooms, facilities, etc.)
High-quality photos, videos, and virtual tours
Interactive site plan and unit availability status
Price lists and payment scheme information
Lead capture and inquiry forms
Schedule site visits and property viewings
Mortgage/financing calculator
Agent contact and WhatsApp integration
Promotions and featured projects
Location maps and nearby amenities
Customer account for saving favorite units
Admin dashboard for managing projects, units, pricing, and leads
Analytics dashboard to track views, inquiries, and conversions

Target Users:

Property developers
Real estate sales teams
Property agents
Home buyers and investors

Value Proposition:
A centralized digital platform that helps developers present properties professionally, generate qualified leads, and streamline the sales process from discovery to booking.
